# Netflix-BI-Dashboard

This project showcases my ability to transform raw data into actionable insights using Power BI. I designed an interactive dashboard that analyzes Netflix’s content library, helping users explore trends in:

🎬 Genre distribution – Which categories dominate the platform

🌍 Country-wise content availability – Regional strengths and gaps

📈 Yearly release trends – Growth patterns in movies vs. TV shows

⭐ Ratings & audience preferences – What viewers engage with most

🔧 Tools & Techniques

Power BI for visualization and interactivity

Data cleaning & modeling to ensure accuracy

DAX measures for advanced calculations

Storytelling dashboards that make insights easy to understand
